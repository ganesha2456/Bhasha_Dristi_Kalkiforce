Bhasha Dristi is a smart mobile application that enables users to read, transliterate, and understand text written in different Indian scripts.
It helps travelers, students, and citizens easily read signboards, notices, books, and handwritten text across languages without knowing the script.

For example:

A person from Andhra Pradesh can read Punjabi signboards.

A visitor in Kerala can read Malayalam boards even if they only know Hindi or English.

Students can instantly convert regional scripts into Roman or their native script.

This is transliteration (script conversion), not translation, ensuring original phonetic(sound/pronounciation) is preserved.
